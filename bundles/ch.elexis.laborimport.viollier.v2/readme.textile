h2. Einleitung

Die gemessenen Labordaten werden von Viollier in ein HL7-Meldung (ORU^R01) verpackt und mit Hilfe von OpenMedical
auf den Elexis-Rechner geladen (MedTransfer). Diese Daten können dann in Elexis importiert werden. 
Der Import wird manuell gestartet.
Nutzt zum Parsing der HL7 Nachrichten das HAPI via ch.elexis.hl7.v2x
