h2. Anforderungen

Diese Plug-In benötigt Elexis in Version >2.1.5. Das mythic Blut-Analysegerät und die Elexis-Instanz die die Daten importiert müssen über eine Netzwerkverbindung zueinander verfügen.

h2. Installation

Die Installation des Plug-Ins selbst erfolgt mittels des Konfigurators, es müssen keinerlei spezielle Massnahmen ergriffen werden.


h2. Konfiguration

Mapping File, nur Wertepaare eintragen für die real ein Ziel-Laboritem existiert. Gibt es einen Schlüssel ohne Wert führt das zu einem Fehler.
